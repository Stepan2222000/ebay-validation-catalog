# junk_filter

Самодостаточный пакет мусор-фильтра: лёгкая модель (MiniLM, 22M, ~90 МБ) отделяет
настоящие авто/мото/водные запчасти от ширпотреба (Funko, чехлы, тонер, постеры).

## Состав
- `model/` — обученная модель (синтетика + реальные тайтлы)
- `classifier.py` — загрузка + инференс (CPU, ~1.4 мс/тайтл, старт ~1.7 с)
- `__init__.py` — публичный API

## API
```python
from junk_filter import is_junk, part_scores, classify_units
is_junk("Funko Pop! Vinyl Figure #24")        # -> True
is_junk("In-Tank Fuel Pump Sea-Doo 275500962")# -> False
part_scores([t1, t2, ...])                     # -> [P(деталь), ...]
classify_units(units)                          # -> {ключи мусора}, вердикты не трогает
```

## Подключение к валидатору (один хук)
В `validation.py` валидатор импортирует пакет и вызывает `classify_units`.
Сам ML в пакете — `src/validator` остаётся чистым.

## Метрики (на 150 реальных, в обучении их нет)
- точность 99%, убитых деталей 0 (порог 0.15), мусора поймано 34/36
- на полном каталоге: ~1 ложное срабатывание, ~22% PDP-кандидатов парсера — мусор

## Переобучение
`/tmp/retrain.py` — синтетика (`/tmp/gen_synth.py`) + реальные тайтлы из живой базы.
Порог: env `JUNK_FILTER_THRESHOLD` (default 0.15).
